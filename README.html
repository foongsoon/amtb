阿弥陀佛简介

阿弥陀佛（梵语：Amitābha），又称无量光佛、无量寿佛，是西方极乐世界的教主。在佛教净土宗信仰中，持念阿弥陀佛名号，可蒙佛接引往生西方净土，永离轮回之苦。

阿弥陀佛在成佛前曾发四十八大愿，其中第十八愿言："设我得佛，十方众生，至心信乐，欲生我国，乃至十念，若不生者，不取正觉。"此即净土宗"念佛往生"之根本依据。

常念"南无阿弥陀佛"六字洪名，可消除罪业、增长福慧、得佛护佑，临终蒙佛接引，往生极乐世界，成就无上佛道。

净土经典

"从是西方，过十万亿佛土，有世界名曰极乐。其土有佛，号阿弥陀，今现在说法。"
——《佛说阿弥陀经》
"若有善男子、善女人，闻说阿弥陀佛，执持名号，若一日、若二日、若三日、若四日、若五日、若六日、若七日，一心不乱。其人临命终时，阿弥陀佛与诸圣众，现在其前。是人终时，心不颠倒，即得往生阿弥陀佛极乐国土。"
——《佛说阿弥陀经》
"设我得佛，十方众生，发菩提心，修诸功德，至心发愿，欲生我国，临寿终时，假令不与大众围绕现其人前者，不取正觉。"
——《佛说无量寿经》· 第十九愿

念佛方法

1. 端身正坐，放松身心

2. 随呼吸默念或出声念"南无阿弥陀佛"

3. 心念耳闻，字字分明

4. 每日坚持，持之以恒

愿以此功德，庄严佛净土。上报四重恩，下济三途苦。

若有见闻者，悉发菩提心。尽此一报身，同生极乐国。
